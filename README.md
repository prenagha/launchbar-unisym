# Unicode Symbols for LaunchBar

A Collection of unicode symbols in [LaunchBar 6](http://obdev.at/products/launchbar).

- Inspired by [copypastecharacter.com](http://copypastecharacter.com)
- Over 9000 glyphs
- Live search
- Easy paste 
- Unicode 7.0.0
